# plane

vue3+pixi.js实现飞机大战

## 页面
- 开始页面
- 游戏页面
- 结束页面

## 实体
- 我方飞机
    - 键盘控制移动
    - 空格攻击
- 我方子弹
    - 自动向上移动
- 敌方飞机
    - 自动向下移动
- 地图
    - 地图向下滚动
- 碰撞检测
    - 我方飞机和敌方飞机碰撞 >> 游戏结束
    - 我方子弹和敌方飞机碰撞
